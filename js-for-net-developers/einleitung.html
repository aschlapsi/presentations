JavaScript ist die Sprache des Web.

  • 1995 von Brendan Eich (Netscape) als LiveScript entwickelt.
  • Ursprüngliches Ziel: Validierung von Formulareingaben
  • Kooperation mit Sun Microsystems: Interaktion mit Java-Applets, Umbenennung in JavaScript
  • 1997 von ECMA standardisiert: ECMAScript

Aktuelle Versionen:

Anwendungsmöglichkeiten

  • Client-seitige Webprogrammierung (Browser)
  • Server-seitige Webprogrammierung
    • Node.js - http://nodejs.org
    • RingoJS - http://ringojs.org
  • Smartphone-Anwendungen
    • Mobile Web Apps
    • Native Apps
      • PhoneGap - http://www.phonegap.com
      • Titanium Appcelerator - http://www.appcelerator.com
  • Browser-Erweiterungen: Firefox, Chrome, Safari
  • Windows: Scripting Host, Windows Gadgets, Windows 8?
  • In eigene Programme einbetten: IronJS, Jurassic, Rhino, V8

JavaScript ist interessant

  • Einflüsse von: C, Self, Scheme, Perl, Python, Java
  • Dynamisch und schwach typisiert
  • Objekt-orientiert, aber klassenlos: Vererbung via Prototypen
  • Unterstützt mehrere Programmierparadigmen:
    • prozedural
    • objekt-orientiert
    • funktional: first-class functions, nested functions, closures
  • Prototyp-basiert: Prototypen, Funktionen als Objektconstructor, Funktionen als Methoden
