Node.js

  • Zur Zeit sehr populär
  • Bereits ein eigenes Ökosystem entwickelt
  • Microsoft unterstützt Portierung auf Windows
  • Eigener Package-Manager (wie Nuget, Rubygems, ...): Node Package Manager - npm (http://npmjs.org)
  • Verwendet V8 von Google als JavaScript-Engine

RingoJS

  • Nachfolger von Helma
  • Orf.at-Network teilweise mit Helma realisiert
  • Verwendet Rhino von Mozilla als JavaScript-Engine
  • Rhino ist in Java entwickelt
  • Von Helma/RingoJS kann man auf Java-Code zugreifen
